# Launch Plan

ScenarioSeed will not get stars passively. The first target is not GitHub search; it is a clear launch post that sends the right people to the repo.

## One-Sentence Pitch

Faker creates fake fields. ScenarioSeed creates fake businesses.

## Short Post

I built ScenarioSeed, a small CLI that turns a Prisma schema into business-state seed data.

Instead of generating 100 random users, it generates scenarios like:

- trial expiring
- past-due enterprise account
- usage over quota
- payment failed
- refunded order
- delayed shipment

It outputs a scenario brief, JSON pack, and Prisma seed file.

Repo: https://github.com/Oxygen56/scenarioseed
